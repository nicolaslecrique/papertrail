Unlike partials/, which are whole fragments a route returns directly, these are macros imported into a page with `{% from "components/ui.html" import ... %}`. They exist to keep the card shell, form inputs, and alert markup in ONE place so the pages stay short and can't drift apart (jscpd guards against the copy- paste creeping back). Every daisyUI class used here is a plain literal so the Tailwind template scan still sees it. #} {% macro card(title=none) %}
