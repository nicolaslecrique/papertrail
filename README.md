# papertrail

Tool to study the manipulation of science by lobbies

input a paper, scientist, journal, and we will evaluate the biases surrounding it.

## Development

See [AGENTS.md](AGENTS.md) for the tech stack, repo layout, and the quality gate
(`./scripts/check.sh`) that every change must pass.

To run several coding agents at once, each fully isolated in its own devcontainer and
database, see [docs/multi-agent-dev-setup.md](docs/multi-agent-dev-setup.md).
